# Bye Bye Alice AI

Простое браузерное расширение, которое убирает блоки и вкладки с нейросетью "Алиса" из поисковой выдачи Яндекса.

## Возможности

-   **Удаляет нейроответы:** Полностью скрывает большие информационные блоки от "Алисы" в начале страницы.
-   **Убирает вкладку "Алиса":** Скрывает кнопку-вкладку "Алиса" из навигационной панели.
-   **Простое управление:** Включайте и отключайте блокировку одним кликом по иконке расширения.
-   **Индикатор состояния:** Иконка расширения меняет цвет, показывая, активна ли блокировка в данный момент.

## Установка

### Из исходного кода

1.  Скачайте этот репозиторий как ZIP-архив (`Code` -> `Download ZIP`) и распакуйте его.
2.  Откройте ваш браузер (Chrome, Edge и т.д.) и перейдите на страницу расширений (`chrome://extensions`).
3.  Включите "Режим разработчика" (обычно это переключатель в углу страницы).
4.  Нажмите кнопку "Загрузить распакованное расширение" и выберите папку, которую вы распаковали на первом шаге.
